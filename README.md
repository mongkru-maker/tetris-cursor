# 테트리스 (교육용)

HTML, CSS, JavaScript만 사용하는 간단한 테트리스 게임 프로젝트입니다.

## 실행 방법

1. 이 폴더를 파일 탐색기에서 연다.
2. `index.html` 파일을 더블 클릭하거나, 브라우저로 드래그한다.
3. **시작** 버튼을 클릭해 게임을 시작한다.

### VS Code / Cursor에서 실행 (선택)

Live Server 확장을 사용하는 경우:

1. `index.html`을 연다.
2. 우클릭 후 **Open with Live Server**를 선택한다.

## 파일 구조

```
tetris-cursor/
├── index.html   # 게임 화면 구조
├── style.css    # 스타일
├── script.js    # 게임 로직
└── README.md    # 이 파일
```

## 현재 상태

- 10×20 CSS Grid 보드
- 7종 테트로미노 (I, O, T, S, Z, J, L) 렌더링
- 자동 낙하 및 충돌 판정 (`canMove`)
- 키보드 조작 (이동, 회전, 소프트/하드 드롭)
- 라인 삭제 및 점수 계산
- 게임 오버 표시 및 재시작

## 점수 규칙

| 삭제한 줄 수 | 점수 |
|---|---|
| 1줄 | 100 |
| 2줄 | 300 |
| 3줄 | 500 |
| 4줄 | 800 |

## 게임 오버

새 블록이 스폰 위치에 놓일 수 없으면 게임 오버가 됩니다. 보드 위에 **게임 오버** 메시지가 표시되고, **재시작** 버튼으로 다시 플레이할 수 있습니다.

## 조작법

| 키 | 동작 |
|---|---|
| `ArrowLeft` (←) | 왼쪽 이동 |
| `ArrowRight` (→) | 오른쪽 이동 |
| `ArrowDown` (↓) | 한 칸 빠르게 내리기 |
| `ArrowUp` (↑) | 회전 |
| `Space` | 즉시 낙하 (Hard Drop) |

모든 조작은 `canMove()` 충돌 판정을 통과할 때만 적용됩니다. 회전 후 벽이나 고정 블록과 충돌하면 회전은 취소됩니다.

## 플레이 방법

1. **시작** 또는 **재시작** 버튼으로 게임을 시작한다.
2. 방향키와 Space로 블록을 조작한다.
3. 가로줄을 가득 채우면 줄이 삭제되고 점수가 올라간다.
4. 보드 상단까지 블록이 쌓이면 게임 오버가 된다.
